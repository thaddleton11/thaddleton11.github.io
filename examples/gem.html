GEM, or Glasgoes Event Management System, is the in-house administration software for all event registrations, apps and websites.

GEM is build upon PHP Slim 2 and uses raw MySQL queries to access backend data. AdminLTE admin panel template has been used as this is based on Bootstrap 3. The system incorporates a CMS, user administration and invoices/payments for delegate bookings and accounting information.

The project developed on by myself and two other developers and, therefore, using Git is extremely hand. Each of us tend to development particular sections but it is ensured we keep up-to-date with each others significant changes so we are able to code throughout the system. The Glasgows Event App is served from this system, using the CMS and user information to control and update user contain.
